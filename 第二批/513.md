# [css] 第513天 分离图层做动画有什么优点？

- 每个图层独立渲染，元素的更新不会影响到其他图层，减少了不必要的重排和重绘
- 可以实现更复杂的动画效果
- 可以更容易利用gpu进行硬件加速

# [js] 第513天 说说你对RAIL性能评估模型的了解

他是一个评估web应用性能的模型
- R，response响应，用户输入网页快速给出结果的过程，如点击按钮、切换表单等，应在100ms内做出响应
- A，animation动画，如用户滚动网页、拖到刷新、页面动画等，动画效果应流畅，无卡顿现象
- I，idle空闲，当用户无操作时可以利用这段时间执行一些后台操作
- L，load加载，网页加载的速度应越快越好
